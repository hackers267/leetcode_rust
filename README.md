# 力扣的Rust版本实现

使用Rust来实现力扣算法的主要出发点一来是熟悉常用的算法，二来是熟悉Rust的常用语法。同一个题目可能有不同的算法实现，不同的算法实现会通过版本来控制。一次的提交内容会涉及到一个题目的算法实现。同一个题目的不同实现会通过重构的方法来实现。
